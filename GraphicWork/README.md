# Графическая работа: Аквариум с рыбками

## Описание проекта
Этот проект представляет собой симуляцию аквариума с использованием библиотеки GLUT (OpenGL Utility Toolkit) на C++.
В симуляции присутствуют плавающие рыбки, пузырьки воздуха и водоросли.

## Функциональность

### Основные элементы:
1. **Рыбки**
   - Разноцветные рыбки разного размера.
   - Плавная анимация плавания (движение хвоста и плавников).
   - Рыбки плавают в разных направлениях и разворачиваются у границ экрана.
   - Рыбки периодически выпускают пузырьки воздуха.

2. **Пузырьки воздуха**
   - Поднимаются со дна и от рыбок.
   - Имеют эффект покачивания при всплытии.
   - Прозрачные с бликами.

3. **Окружение**
   - Градиентный фон (вода).
   - Песчаное дно с камнями.
   - Анимированные водоросли, колышущиеся в воде.

## Требования для сборки
- Visual Studio (с поддержкой C++).
- Библиотека GLUT (файлы включены в проект).

## Структура проекта
- `Program.cpp`: Основной файл с исходным кодом.
- `include/glut.h`: Заголовочный файл GLUT.
- `lib/glut32.lib`: Библиотека импорта GLUT.
- `bin/glut32.dll`: Динамическая библиотека GLUT (должна быть рядом с .exe).

## Как запустить
1. Откройте файл `GraphicWork.slnx` или `GraphicWork.vcxproj` в Visual Studio.
2. Соберите проект (Build -> Build Solution).
3. Запустите проект (Debug -> Start Debugging).

Если возникнет ошибка с отсутствием `glut32.dll`, скопируйте файл `bin/glut32.dll` в папку с исполняемым файлом (обычно `x64/Debug/` или `Debug/`).
